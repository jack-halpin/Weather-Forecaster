Welcome to Weather Forecaster.

This site uses the OpenWeatherMap API to bring you 5 day forecasts from up to 200,000 cities across the world. To get started, Simply enter the city you're interesting in, the units you want to use, the number of days you want the forecast for and the Weather Forecaster will get you results in an instant.

How to Search.

If you'd like to search for a specific location then simply place the name of the city and country code seperated by a comma in the search bar above. For example, to get info on Dublin, Ireland enter 'Dublin,ie'.
